# swingdrag

What's swingdrag?\
swingdrag is a jQuery UI plugin, which adds a nice swing effect to the jQuery UI drag function.

# Version
v0.5.0

# Install
```
npm install swingdrag
```

# Usage
## HTML
```html
<script src="jquery.ui.swingdrag.min.js"></script>

<div id="container"></div>
```
# JavaScript
```js
$("#container").swingdrag();
```
